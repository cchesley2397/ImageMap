# ImageMap
Generating maps and timelines from image metadata.

## Installation


## Dependencies


#### Map
**node.js**

**react.js**
`npm install --save react react-dom`

**babel.js**
`npm install --save-dev babel-cli`

## Usage
**Starting the server** (Requires node)

Navigate to \src\main\web\js, and run `node server.js`.
